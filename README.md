# (S.C.A.M.) Script de Configuración Automática de Marcos

Un script para Debian 13 que ofrece una replica mi escritorio, pensado para el que quiera una experiencia "window manager" en Wayland.

Actualmente se puede elegir entre sway o niri.

Este escritorio esta diseñado siguiendo el estilo minimalista, esto quiere decir que se prioriza el uso de pequeñas herramientas de la terminal en contrapunto a interfaces gráficas.

Esto se traduce en el eso de un menú estilo dmenu (en este caso tofi) para varias funciones del sistema, como el control de la sesión, gestor de contraseñas, etc.

Se han creado herramientas adicionales para este escritorio, como un selector de fondos de pantalla, notificaciones de volumen o brillo y mucho más, estos vienen incluidos en la instalación pero se pueden encontrar en mi repositorio de scripts.
